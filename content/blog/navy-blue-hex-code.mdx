# Navy Blue Hex Code: #000080 — Shades, Palettes & Copy-Paste Codes

The canonical **navy blue hex code** used across CSS references and design tools is **#000080** — full-intensity blue at half luminance in RGB terms, which reads as a deep maritime blue on screens. Paired with **RGB(0, 0, 128)** and **HSL(240, 100%, 25%)**, it anchors trust-heavy interfaces: banking dashboards, government portals, and clinical systems where users expect sobriety and legibility.

Navy is not interchangeable with “dark blue” in every brand system. Designers often reserve **#000080** for literal “navy” while using cooler near-blacks (for example **#0F172A**) for text and chrome. Document your tokens so developers do not silently substitute one for the other.

## Palette pairings

- **Navy + Gold (#000080 + #FFD700)** — classic luxury; gold as thin rules or icons, not large fill behind small text.
- **Navy + White (#000080 + #FFFFFF)** — clean professional; ensure white type on navy passes WCAG for your exact font size.
- **Navy + Coral (#000080 + #FF6B6B)** — modern energetic; coral for alerts and secondary actions.
- **Navy + Light Gray (#000080 + #F5F5F5)** — minimal corporate; gray fields with navy type and navy primary buttons.

## Navy blue in UI and web design

Finance products use navy headers with off-white content wells so dense tables stay readable. Government sites pair navy with high-contrast neutrals for accessibility mandates. Healthcare interfaces layer navy navigation above pale blue-green or gray surfaces to signal calm without clinical coldness. Always run final pairs through a **contrast checker** before launch.

## Navy blue in branding

Brands associated with navy signaling authority and longevity include financial institutions, insurers, and heritage retailers. Navy lets photography and product color pop while the brand frame stays restrained.
